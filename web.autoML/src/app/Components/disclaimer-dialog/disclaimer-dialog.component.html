Disclaimer

1. The upper limit for file size accepted by this system is approximately 10_000 x 100 columns (no more than 8MB). 2. This system performs regression tasks only - if you are trying to predict the class of a set of data points, this is not for you. 3. No categorical data is accepted - if you would like to include a categorical variable, you must encode it numerically. 4. Ensure that you are not uploading sensitive data - it will be temporarily saved to our servers. 5. You can only have one target variable in the file you upload, and it must be in the furthest column to the right. 6. You can include or exclude headers for your data - the system will ignore them if they are included. 7. Your data must not have any nans or missing values, as they will cause the system to fail.
